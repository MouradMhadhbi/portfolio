Interface d’administration

Création d’un tableau de bord permettant la gestion d’utilisateurs, des rôles et du contenu d’un site web.

Ce projet a été développé avec Node.js pour le backend et MySQL pour la base de données. L’interface offre la possibilité d’ajouter, modifier ou supprimer des utilisateurs et articles.

Objectif : concevoir une plateforme d’administration sécurisée, simple à utiliser et connectée à une API interne.
